\section{IAP and SSBA Relationship}
	SSBA looks like this:

	\[ y_{SSBA} = \Re \left( z^{n} \right) \]
	\[ y_{SSBA} = \Re \left( (|z|e^{i\arg(z)})^{n} \right) \]
	\[ y_{SSBA} = |z|^{n}\cos(n\arg(z)) \]

	IAP looks like this:

	\[ y_{IAP} = |z|\cos(n\arg(z)) \]

	Thus:

	\[ y_{SSBA} = |z|^{n-1}y_{IAP} \]
